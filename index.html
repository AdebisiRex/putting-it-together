Git Commands

  • GIT INIT: initialize a new repoository for your project
  • GIT ADD [filenam]: Add a single changed file
  • GIT ADD .: Add a all changed file
  • GIT STATUS: Shows a list of staged files
  • GIT BRANCH -M [NEW_BRANCHNAME]: used to rename branch
  • GIT COMMIT -M '[commit - message]"": creates code checkpoint
